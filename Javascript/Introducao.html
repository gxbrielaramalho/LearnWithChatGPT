O que é Javascript?

JavaScript é uma linguagem de programação que permite adicionar interatividade e dinamismo às páginas web. Com ele, você pode manipular o conteúdo exibido, responder a eventos como cliques de botão, validar formulários, e muito mais.

Como Adicionar JavaScript a uma Página HTML

Há três maneiras principais de adicionar JavaScript a uma página HTML:

JavaScript Inline: Você pode adicionar JavaScript diretamente dentro de um atributo de um elemento HTML, como onclick ou onmouseover. Isso não é muito recomendado para código complexo, mas funciona para pequenas interações.

Você pode adicionar JavaScript diretamente dentro de um atributo de um elemento HTML, como onclick ou onmouseover. Isso não é muito recomendado para código complexo, mas funciona para pequenas interações. 

<button onclick="alert('Olá, mundo!')">Clique aqui</button>

JavaScript Interno: Você pode incluir JavaScript dentro de uma tag <script> no próprio documento HTML, geralmente dentro do <head> ou no final do <body>. 

<html>
 <head>
  <script>
    console.log('Olá, mundo!');
  </script>
 </head>
  <body>
    <p>Veja o console para a mensagem!</p>
  </body>
</html>
